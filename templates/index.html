📊 Верхние карточки
Общая статистика по последним клипам и рекламе за 30 дней — просмотры, лайки, расход бюджета.
↻ Обновить данные
Подгружает свежую статистику из ВКонтакте без перезагрузки страницы.
📋 Таблица видео
Последние клипы группы. Столбец 💬 / 👥 — общее число комментариев / уникальных авторов. Нажми на название видео — откроется в ВКонтакте.
📣 Реклама в таблице
Выбери из выпадающего списка кампанию которая продвигала это видео — покажется статистика: показы, клики, CTR, расход.
📝 Сценарий
Кнопка в последней колонке открывает поле для текста или сценария видео. Синяя иконка — сценарий уже добавлен. Claude использует его при анализе.
✦ Анализ Claude
ИИ анализирует статистику всех видео и даёт конкретные рекомендации — что снимать, когда публиковать, что работает лучше.
💬 Задать вопрос
Поле рядом с кнопкой «Спросить» — любой вопрос по данным. Например: «Какое видео дало лучший CTR?» или «Предложи тему для следующего клипа».
